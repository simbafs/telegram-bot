# My Telegram Bot

# Start

```
TELEBOT_TOKEN=<token> make dev
```

# Features

## ping
`/ping`
Pong!

## counter

`/counter`  
![counter](./img/counter.png)

## Pixiv
`/pixiv <novel id>`
get novel by id, then translate to Taiwanese Chinese

or use can send a link to the bot to trigger this command
